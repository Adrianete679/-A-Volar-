# -AVolar-
## ¿En que consiste el proyecto?
Este proyecto consiste en programar un dron de la marca "TELLO" para que haga una especie de circuito pero dentro del instituto, tiene que hacer un simbolo de infinito al través de dos columnas. Este proyecto se hace en grupo, aunque cada uno programa lo que debe de hacer el robot, que cada grupo tiene uno distinto, ya que cada uno empezará mirando para un lugar distinto.
## Descripción del dron DJI TELLO
1. Tiene 4 hélices en cada esquina del dron, formando un cuadrado. Cada una de estas hélices tiene un motor.
2. En la parte frontal tiene una especie de circulo con luz el cual parpadea cambiando de color indicando el estado en el que se encuentra el dron, siendo el verde para bien; amarillo para regular y rojo para mal. También está el color azul indicando que está cargando y cuando se está conectando
3. Tiene una camara para poder ver lo que ve el dron y asi saber donde está o que es lo que pasa por ahí.
4. En el lado derecho tiene un botón para encender y apagar.
5.  
6. Antenas 7. Sistema de posicionamiento visual
8. Batería de Vuelo 9. Puerto Micro USB 10. Protectores para las hélices
